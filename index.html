Tentang Aplikasi :

Aplikasi Analisis W-NFX adalah alat analisis trading yang membantu menganalisis pasar dan membuat keputusan trading lebih akurat. Aplikasi ini menggunakan teknologi canggih dan algoritma kompleks untuk mendeteksi pola chart dan memberikan sinyal trading akurat. Untuk menggunakan aplikasi ini, pastikan Anda menggunakan garis horizontal warna putih, memperbesar setengah chart, dan memiliki DPI tidak melebihi 600. Aplikasi ini hanya bekerja pada pair XAUUSD dan menggunakan Screen Mode Real-time, jadi pastikan perangkat Anda kompatibel.

1. Atur UI Pada Metatrader :

Pertama-tama Kamu harus atur untuk warna dari UI metatrader kamu, Agar Alat bisa mendeteksi dengan jelas

1. OFF/MATIKAN GRID

Disini Untuk Mematikan Grid pergi ke setelan, cara nya tap satu kali pada chart, lalu tekan logo seperti garis lalu matikan

2. Atur Warna/Color

Dipengaturan sebelum nya, geser ke bawah lalu klik warna

Ubah 2 Warna diatas menjadi hitam, Latar Belakang Dan Latar Depan.

Ubah Jadi Warna Hitam Lalu Klik OK Dan Simpan.

Cara Set Up Untuk Analisis System :

Disini kamu akan Mengatur/Menata Garis Horizontal Untuk System di MetaTrader

Set UP Garis Horizontal

Klik Satu kali pada Chart, untuk memunculkan opsi Objek

Klik Garis Yang muncul pada chart, Lalu klik logo pensil (edit).

Lalu Klik Bulat itu, lalu Warna nya Menjadi Putih Terang, Lalu Klik Simpan.

Pemetaan Candle Pada Chart :

Jika Semua Sudah Diatur Saatnya Set UP Candle Price Nya. Kamu harus menandai HLC(High Low Close) Pada 1 Candle Yang Tidak Bergerak(Sesudah Harga Bergerak Pada Candle Selanjutnya)

Seperti Contoh Ini :

Untuk Garis Usahakan Jangan Menempel atau Menyatu, Itu Bisa Membuat Sistem Error dan NotFound, Jadi Usahakan Terpisah dan Label Harga Jelas. Seperti Ini :

jika semua set up telah kamu lakukan, maka selanjutnya mengaktifkan W-NFX terminal system klik start pada aplikasi, maka akan muncul bubble dengan tombol Start/Play dan Close

untuk memulai analisis (jika semua set up dan siap analisis kamu telah ada) sesuaikan chart dengan stabil, lalu tekan play dan terminal system mulai analisis dan memperlihatkan hasil untuk entry.l

Jika Terminal Menyatakan HLC NotFound, Perbesar Chart atau Set Up kamu dengan Jelas agar System dapat Mendeteksi

Setelah HLC ditemukan, kamu bakal diminta untuk memasukan angka ask atau bid (garis biru yg bergerak cepat, tapi untuk memasukan angka kamu bisa memasukan angka yg dekat dan tidak jauh dengan angka bid atau ask nya)

fungsi ask bid ini untuk memberi pertimbangan entry. sistem mendeteksi dan memperhitungkan entry dari angka ask atau bid saat ini

kesimpulan nya ask dan bid itu angka harga XAUUSD saat ini

baiklah, jika setup dan cara entry yang telah kamu ketahui, silahkan memulai industri trading kamu dengan alat analisis otomatis ini.
